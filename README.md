# Прогнозирование движения космических объектов

Этот репозиторий содержит программу на Python для моделирования и прогнозирования движения космических объектов в околоземном пространстве. Проект учитывает гравитационное притяжение Земли с учетом второй зональной гармоники геопотенциала (J2), использует метод Рунге-Кутты 4-го порядка для численного интегрирования уравнений движения и преобразует декартовы координаты и скорости в кеплеровы элементы орбиты.

Проект разработан в рамках научного исследования, связанного с мониторингом и прогнозированием орбит космических объектов.

## Описание

### Основные компоненты:
- **rk_orb.py**: Основной скрипт для интеграции орбиты. 
  - Реализует метод Рунге-Кутты для решения дифференциальных уравнений движения.
  - Учитывает центральное гравитационное поле Земли и возмущения от J2.
  - Преобразует результаты в кеплеровы элементы и строит график (например, аргумента перигея ω).
- **rv_to_kepler.py**: Модуль для преобразования векторов положения (r) и скорости (v) в кеплеровы элементы орбиты:
  - Наклонение (i)
  - Долгота восходящего узла (Ω)
  - Эксцентриситет (e)
  - Аргумент перигея (ω)
  - Истинная аномалия (ν)
  - Большая полуось (a)

Проект демонстрирует эволюцию орбитальных элементов под влиянием нецентральности гравитационного поля Земли.

### Теоретическая основа:
- Уравнение движения: r'' = -μ / r^3 * r + a_J2, где a_J2 — возмущение от J2.
- Метод интегрирования: Рунге-Кутта 4-го порядка для точного численного решения.
- Тестирование: Включено тестирование точности метода на экспоненте и сравнение с известными данными (например, из учебников по астродинамике).

## Использование

Запустите основной скрипт:
```
python rk_orb.py
```

- В скрипте заданы начальные условия (положение и скорость в декартовых координатах).
- Интеграция проводится на интервале времени от 0 до 108000 секунд с шагом 10 секунд.
- Вывод: График изменения аргумента перигея (ω) со временем. Другие элементы можно визуализировать, раскомментировав соответствующие строки.

### Пример изменения параметров:
В файле `rk_orb.py` вы можете изменить:
- Начальное состояние `s_0` (положение и скорость).
- Интервал интегрирования `t_0`, `t_1`.
- Шаг интегрирования `d`.
- Визуализируемый элемент (i, Ω, e, ω, ν, a).

Для тестирования преобразования RV в Kepler:
В файле `rv_to_kepler.py` раскомментируйте тестовый пример и запустите:
```
python rv_to_kepler.py
```

## Результаты

- Программа прогнозирует траекторию и эволюцию орбитальных элементов.
- Учет J2 приводит к вековым возмущениям, таким как прецессия узлов и перигея.
- Пример вывода: График ω(t), показывающий линейный рост аргумента перигея из-за J2.
